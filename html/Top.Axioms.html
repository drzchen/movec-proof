Axioms for primitives

(accessMem, storeMem, umalloc, ufree, ucreateFrame, udestroyFrame)
The memory Mem is defined only for addresses that have been allocated to the program by the C runtime. The C runtime provides primitive operations for accessing the memory: accessMem, storeMem, umalloc, ufree, function frame allocation ucreateFrame and function frame deallocation udestroyFrame. Rather than committing to a particular implementation of these primitives, our formalism axiomatizes properties that any reasonable implementation should satisfy.

Axiom validAddressRange :
  0 < funAddress ∧ funAddress ≤ minAddress ∧
  minAddress ≤ baseAddress ≤ maxAddress.

An allocated status node in the status node table must be previously available for allocation.
Axiom getFreshSEntry__inversion : ∀ ST sa,
  getFreshSEntry ST = Some sa →
  ST sa = None ∧ sa ≠ 0 ∧ sa ≠ f_sa ∧ sa ≠ g_sa.

The axioms of accessMem and storeMem state simple properties like: ``reading a location after storing to it returns the value that was stored'', ``storing to a location does not affect any other location'', ``storing a value to a location is valid iff storing another value to it is valid'', ``reading a location is valid iff storing to it is valid''.
Note that accessMem and storeMem fail if they try to access unallocated memory.

Axiom storeMem__accessMem : ∀ M loc val M',
  
  storeMem M loc val = Some M' →
  
  accessMem M' loc = Some val ∧
  
  (∀ l v, l ≠ loc →
    accessMem M l = Some v → accessMem M' l = Some v) ∧
  
  (∀ l,
    accessMem M l = None → accessMem M' l = None)
  .

Axiom storeMem__uniqBehavior : ∀ M l v v',
  ((∃ M', storeMem M l v = Some M') →
   (∃ M', storeMem M l v' = Some M')) ∧
  (storeMem M l v = None → storeMem M l v' = None)
  .

Axiom validAccessMem__validStoreMem : ∀ M l v,
  (∃ v', accessMem M l = Some v') ↔ (∃ M', storeMem M l v = Some M')
  .

The axioms of umalloc and ufree enforce properties like: ``malloc returns a pointer to a memory block that was previously unallocated, and stores this block at the AllocMemBlock table'', ``free only deallocates a memory block stored at the AllocMemBlock table, and removes this block from the AllocMemBlock table'', and ``malloc and free do not alter the contents of irrelevant locations.''
Note that umalloc fails if there is not enough space, and ufree fails if it tries to deallocate a memory block that is not at the AllocMemBlock table.

Axiom umalloc__inversion : ∀ uE n uM' uG' uAMB' uS' b,
  umalloc uE n = Some ((MkuEnv uM' uG' uAMB' uS'), b) →
  ∃ uM uAMB,
  
  uE = MkuEnv uM uG' uAMB uS' ∧
  
  n > 0 ∧ baseAddress ≤ b ∧ b + n ≤ uE.(ustackvar).(utop) ∧
  
  (∀ l v,
     accessMem uM l = Some v → accessMem uM' l = Some v) ∧
  
  (∀ l, l < b ∨ l ≥ b + n →
     accessMem uM l = None → accessMem uM' l = None) ∧
  
  (∀ l, b ≤ l < b + n →
     accessMem uM l = None) ∧
  
  (∀ l, b ≤ l < b + n →
     accessMem uM' l = Some 0) ∧
  
  uAMB' = addAllocMemBlock uAMB b (b + n)
  .

Axiom ufree__inversion : ∀ uE loc uM' uG' uAMB' uS',
  ufree uE loc = Some (MkuEnv uM' uG' uAMB' uS') →
  ∃ uM uAMB b e,
  
  uE = MkuEnv uM uG' uAMB uS' ∧
  
  accessMem uM loc = Some b ∧ uAMB b = Some e ∧
  
  (∀ l, l < b ∨ l ≥ e →
     accessMem uM l = accessMem uM' l) ∧
  
  (∀ l, b ≤ l < e →
     ∃ v, accessMem uM l = Some v) ∧
  
  (∀ l, b ≤ l < e →
     accessMem uM' l = None) ∧
  
  baseAddress ≤ b ∧ e ≤ uE.(ustackvar).(utop) ∧
  
  uAMB' = removeAllocMemBlock uAMB b
  .

The axioms of ucreateFrame and udestroyFrame enforce properties like: ``creating a frame allocates a new frame on the top of the stack'', ``destroying a frame simply removes the latest frame from the stack'', and ``creating and destroying a frame do not alter the contents of irrelevant locations.''
Note that ucreateFrame fails if there is not enough space, and udestroyFrame fails if it tries to destroy a frame from an empty stack.

Axiom ucreateFrame__inversion : ∀ uE fr uE',
  ucreateFrame uE fr = Some uE' →
  ∃ uF',
  
  uE.(uglobalvar) = uE'.(uglobalvar) ∧
  uE.(uamb) = uE'.(uamb) ∧
  
  head uE'.(ustackvar).(uframes) = Some uF' ∧
  uFrame2frame uF' = fr ∧
  uStackVar2frame uE'.(ustackvar) = fr ∧
  (∀ v t, fr v = Some t →
     ∃ l, lookUpuStackVar uE'.(ustackvar) v = Some (l, t) ∧
               uF'.(ufrom) ≤ l < uF'.(uto)) ∧
  uF'.(ufrom) = uE'.(ustackvar).(utop) ∧
  uF'.(uto) = uE.(ustackvar).(utop) ∧
  tail uE'.(ustackvar).(uframes) = uE.(ustackvar).(uframes) ∧
  baseAddress < uE'.(ustackvar).(utop) < uE.(ustackvar).(utop) ∧
  
  (∀ l v,
     accessMem uE.(umem) l = Some v → accessMem uE'.(umem) l = Some v) ∧
  
  (∀ l, l < uF'.(ufrom) ∨ l ≥ uF'.(uto) →
     accessMem uE.(umem) l = None → accessMem uE'.(umem) l = None) ∧
  
  (∀ l, uF'.(ufrom) ≤ l < uF'.(uto) →
     accessMem uE.(umem) l = None) ∧
  
  (∀ l, uF'.(ufrom) ≤ l < uF'.(uto) →
     accessMem uE'.(umem) l = Some 0) ∧
  
  wfuFrameNoOverlapped uF'
  .

Axiom udestroyFrame__inversion : ∀ uE uE',
  udestroyFrame uE = Some uE' →
  ∃ uF,
  
  uE.(uglobalvar) = uE'.(uglobalvar) ∧
  uE.(uamb) = uE'.(uamb) ∧
  
  head uE.(ustackvar).(uframes) = Some uF ∧
  uF.(uto) = uE'.(ustackvar).(utop) ∧
  tail uE.(ustackvar).(uframes) = uE'.(ustackvar).(uframes) ∧
  
  (∀ l v, l < uF.(ufrom) ∨ l ≥ uF.(uto) →
     accessMem uE.(umem) l = v → accessMem uE'.(umem) l = v) ∧
  
  (∀ l, uF.(ufrom) ≤ l < uF.(uto) →
     ∃ v, accessMem uE.(umem) l = Some v) ∧
  
  (∀ l, uF.(ufrom) ≤ l < uF.(uto) →
     accessMem uE'.(umem) l = None)
  .
